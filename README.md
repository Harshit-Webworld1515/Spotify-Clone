# Spotify Clone

This is a front-end clone of the popular music streaming platform **Spotify**, created using **HTML** and **CSS**.  
The project focuses on replicating the user interface of Spotify, including a sidebar, header, playlist section, and music player layout.

## Features
- Responsive design
- Sidebar navigation
- Playlist and song cards
- Music player layout (UI only, no actual music)

## Technologies Used
- HTML5
- CSS3
- Flexbox and Grid for layout
- Responsive design techniques
